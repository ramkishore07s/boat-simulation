## Compiling & Executing
```
1. mkdir build 
2. cd build
3. cmake ..
4. make
5. ./graphics_assgn1
```
---
## Controls: 
keys|action
-----------|----------------
W/S|canon aim
F|fire
C|change camera view
B|Boost
Space|Jump
Arrow keys|Movement
